HTML

  • <tags>:
    • Open/close: <p>this is a paragraph</p>
    • Self-closing: <img src="pic.png" />
  • Basic Structure: doctype → html → head [title] → body
  • Examples:
    • Regular: div, p, strong, i, br*, h1, hr*, blockquote
    • Irregular: img* [src], a [href], ul → li
    • Forms: form → input* [type], button* [type], textarea, select → option
    • All: [id, class, invention-of-your-choice]

CSS

  • Basic Structure:
    • selector { "attribute" : "value" }
    • e.g. h1 { "color" : "blue" }
  • Selectors:
    • html, .class, #id
  • Attributes/Values:
    • width [0], height [0], padding [0 0 0 0], margin [0 0 0 0], border [0 0 0 0], display [none, block], position [relative, absolute, fixed], color [name, hex], background [name, hex], font-family [list of fonts], font-size [0], border-radius [0 0 0 0]

JavaScript

  • jQuery - Cross-browser JavaScript Library
  • Basic Structure:
    • $('selector').function(parameters);
    • e.g. $('h1').hide();
    • e.g. $('#hi').html('This is some new text');
    • e.g. $('.hello').css('color','green');
    • e.g. $('.hello').mouseover(function() { alert('I am awesome'); });
